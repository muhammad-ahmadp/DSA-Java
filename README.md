# Data Structures and Algorithms

A comprehensive collection of DSA implementations in Java covering fundamental concepts and problem-solving approaches.

## Overview

This repository serves as a learning resource and reference for various data structures and algorithms. It contains clean, well-documented implementations of fundamental DSA concepts that form the foundation of efficient problem-solving in computer science. Each implementation is designed to demonstrate key algorithmic principles and best practices.

The primary goal of this repository is to build a strong foundation in DSA while maintaining code quality and clarity, making it a valuable resource for both learning and interview preparation.

---

## Topics Covered

The repository is organized by major DSA topics, each containing multiple implementations and variations:

*   **Searching Algorithms** - Linear Search, Binary Search, and variations
*   **Sorting Algorithms** - Bubble Sort, Merge Sort, Quick Sort, and more
*   **Arrays and Strings** - Array manipulation and string processing problems
*   **Linked Lists** - Single and Doubly Linked Lists with various operations
*   **Trees and Graphs** - Traversals, BST, and graph algorithms
*   **Dynamic Programming** - Optimization problems and memoization techniques
*   **Recursion** - Recursive problem-solving approaches
*   **And more...** - Additional advanced topics

## How to Run

### Prerequisites
*   Java Development Kit (JDK) 8 or higher installed on your system

### Installation and Execution

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/muhammad-ahmadp/dsa.git
    cd dsa
    ```

2.  **Compile all files:**
    ```bash
    javac *.java
    ```

3.  **Run any specific implementation:**
    ```bash
    java package.ClassName
    ```
